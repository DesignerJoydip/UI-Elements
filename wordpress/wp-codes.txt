Process 2 >> ---------------------------------
create a function in functions.php
function example_enqueue_scripts(){ 
}
add_action('wp_enqueue_scripts','example_enqueue_scripts');
// explain
add_action('wp_enqueue_hook_name','your_enqueue_name');

// style function description
wp_enqueue_style( $handle, $src, $deps, $ver, $media );
Explain:
- $handle >> ID of css file (*)
- $src >> file root (*)
- $deps >> Dependency
- $ver >> version
- $media >> support

example:
if css file is "main.css" then link process is
wp_enqueue_style( 'main', get_template_directory_uri().'/assets/css/main.css', array(), '1.0', 'all' );
add this code inside "{ }" of example_enqueue_scripts function.

final:
function example_enqueue_scripts(){ 
	wp_enqueue_style( 'main', get_template_directory_uri().'/assets/css/main.css', array(), '1.0', 'all' );
}
add_action('wp_enqueue_scripts','example_enqueue_scripts');

// support extarnal script files
wp_enqueue_script( 'script-name', get_template_directory_uri() . '/js/example.js', array(), '1.0.0', true );
Explain:
- true : script show above body clase tag 
- false : inside head tag

function template_meta_box_1_save( $post_id )
{
    // Bail if we're doing an auto save
    if( defined( 'DOING_AUTOSAVE' ) && DOING_AUTOSAVE ) return;
     
    // if our nonce isn't there, or we can't verify it, bail
    if( !isset( $_POST['meta_box_nonce'] ) || !wp_verify_nonce( $_POST['meta_box_nonce'], 'template_meta_box_1_nonce' ) ) return;
     
    // if our current user can't edit this post, bail
    if( !current_user_can( 'edit_post' ) ) return;

    // custom code goes here


}
add_action( 'save_post', 'template_meta_box_1_save' );

function template_taxonomies() {

  $labels = array(
  'name' => 'Services Type',
);
  register_taxonomy('ServiceType', 'services', array(
    'labels' => $labels,
    'rewrite' => array('slug' => 'services'),

    )
  );
}
add_action('init','template_taxonomies');
